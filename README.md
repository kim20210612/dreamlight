# 꿈빛마을

GitHub Pages와 Vercel에서 실행할 수 있는 정적 웹사이트입니다.

## GitHub 업로드

ZIP을 먼저 압축 해제한 다음, 아래 파일 전체를 저장소 최상단에 올리세요.
특히 `monster-battle.mp4`, `shapesxr.mp4`, `forever-free.mp3`를 반드시 함께 올려야 합니다.

- index.html
- house-names.js
- monster-battle.mp4
- shapesxr.mp4
- forever-free.mp3
- vercel.json
- .nojekyll
- .gitignore
- .github/workflows/deploy-pages.yml

GitHub Pages는 `Settings → Pages → Source → GitHub Actions`로 설정합니다.
