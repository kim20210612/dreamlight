# 마을 미리보기

React·Three.js 기반 인터랙티브 정적 웹사이트입니다.

## GitHub 업로드

저장소 최상단에 다음 파일과 폴더가 위치하도록 업로드합니다.

```text
index.html
vercel.json
README.md
.gitignore
assets/
  videos/
    shapesxr.mp4
```

## Vercel 배포

1. GitHub에 새 저장소를 만듭니다.
2. 이 폴더 안의 파일 전체를 저장소 최상단에 업로드합니다.
3. Vercel에서 **Add New → Project**를 선택합니다.
4. 해당 GitHub 저장소를 연결합니다.
5. 별도 빌드 설정 없이 **Deploy**를 누릅니다.

> ZIP 파일 자체를 GitHub 저장소에 올리는 것이 아니라, ZIP 압축을 해제한 뒤 내부 파일 전체를 업로드해야 합니다.
